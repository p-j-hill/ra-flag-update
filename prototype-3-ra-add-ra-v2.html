Tips for adding Reasonable Adjustments

  • Browse the cateogry list to find the ones you want to add by checking the boxes.
  • Or use the input box to type keywords. Suggestons will be shown as you type.
  • Select the one that matches. Then press Add.
  • If nothing matches, or you wish to create a bespoke adjustment, continue typing remembering to press Add when you've finished.
  • You can add additional details or delete Reasonable Adjustments witin the Added Reasonable Adjustments box.
  • When you have added all reasonable adjustments press Next.
  • Browse Reasonable Adjustment categories
